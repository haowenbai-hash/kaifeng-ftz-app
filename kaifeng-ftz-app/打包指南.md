# 开封自贸区官方 APP - 快速打包指南

## 方案一：GitHub Actions 自动编译（推荐，5分钟出 APK）

### 步骤

1. **注册 GitHub 账号**（免费）：https://github.com/signup

2. **创建新仓库**
   - 访问 https://github.com/new
   - Repository name 填写：`kaifeng-ftz-app`
   - 选择 **Public**（免费用户需要公开仓库才能使用 Actions）
   - 点击 **Create repository**

3. **上传项目文件**（方法一：网页拖拽）
   - 进入刚创建的仓库页面
   - 将整个 `kaifeng-ftz-app` 文件夹的**内容**拖拽到浏览器页面
   - 点击 **Commit changes**

4. **等待编译**
   - 上传完成后，自动触发 Actions 编译
   - 点击仓库的 **Actions** 标签页
   - 等待绿色勾勾出现（约 3-5 分钟）

5. **下载 APK**
   - Actions 完成后，点击对应的工作流运行记录
   - 在底部 **Artifacts** 部分点击 `kaifeng-ftz-debug` 下载
   - 解压 ZIP 即得到 `app-debug.apk`

6. **安装到手机**
   - 将 APK 传到手机
   - 在手机设置中允许「安装未知来源的应用」
   - 点击 APK 文件安装

## 方案三：AppsGeyser 在线一键生成（最简单）

> 不需要代码，直接填网址即可生成 APK

1. 访问：https://appsgeyser.com/create/website
2. **Website URL** 填写：`https://zmt.ftz.kaifeng.gov.cn/`
3. **App Name** 填写：`开封自贸区`
4. 上传 Logo 图片（可选）
5. 点击 **Create** → 注册账号 → 下载 APK

**注意**：AppsGeyser 免费版会有广告，如需去广告需付费

## 方案四：本地编译（已安装 Android Studio）

```
1. 打开 Android Studio
2. File → Open → 选择 kaifeng-ftz-app 文件夹
3. 等待 Gradle 同步（首次约 5 分钟）
4. Build → Build Bundle(s)/APK(s) → Build APK(s)
5. APK 位于：app/build/outputs/apk/debug/app-debug.apk
```
